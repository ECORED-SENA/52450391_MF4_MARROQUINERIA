2. Terminación y empaque


Es la última fase del proceso, donde se realiza la limpieza y abrillantado del producto, para empacar posteriormente, ya sea en bolsas o cajas de cartón.

Entre las operaciones básicas que se precisan en la terminación del producto están:

2.1 Herramientas


Las herramientas se utilizan en la confección de los modelos, permitiendo trabajar los materiales utilizados. Algunas de las herramientas utilizadas en la elaboración de moldes, también son utilizadas en la confección de modelos como es el caso de la cuchilla, descrita en el numeral anterior.

Lezna (de 4 filos)

Punzón de hierro o acero con punta aguda, conformada por un mango de madera. Se utiliza de forma manual para perforar el cuero, por donde se pasan los hilos del pespunte.

Estaquillo

Herramienta de punta redonda o roma, que se utiliza para marcar puntos de referencia y líneas guía en la moldería. También se usa para asentar el pegado de piezas curvas.

Agujas

Utilizadas para coser el cuero o textiles. Se encuentran en el comercio para costuras manuales y para coser a máquina.
Las agujas para costuras manuales son agujas guarnicioneras y las agujas para máquina de coser son aquellas que no deben faltar al momento de usar cualquier máquina, pues la costura depende de estas. Así mismo, cada una de ellas necesita una aguja especial y se deben tener en cuenta los tamaños según la máquina de coser.

Destornillador

Utilizados para realizar el cambio de agujas y para efectuar ajustes ocasionales a las máquinas. Se necesitan de diferentes puntas y tamaños.

Caja Bobina

Herramienta usada en la parte inferior de la máquina de coser, que recibe y hace que el carretel con hilo entrelace la puntada.

Martillo

Herramienta utilizada para afirmar el pegado y aplanado de piezas. Para ello, se deben realizar golpes suaves y constantes sobre las piezas. También utilizada para golpear los sacabocados o fistos, colocación de remaches y broches. Se encuentran de cabeza metálica o de fibra.

Mordaza

Elemento de madera en forma de pinza, cuyos brazos sujetan el cuero para perforarlo o coserlo.

Compás de puntas

Sirve para realizar trazos exactos cuando hay curvas. Es útil para trasladar medidas o puntos y para marcar las ventajas de costuras.

Sacabocados o fistos

Instrumento de hierro con boca hueca y cortes afilados, que sirve para taladrar. Los hay en diferentes figuras: trébol, hojas, círculos, corazón, etc.

Matacantos

Es una herramienta de corte que permite dar un mejor acabado. Se utiliza para cortar en bisel los bordes de las piezas de cuero.

Repujadores o mateadores

Herramientas para realizar grabados en bajo relieve en cuero curtido al vegetal. Se usa para decorar.

Fibra

Fibra de polipropileno, usada para perforar con los sacabocados. El empleo de esta no deja dañar el corte de los mismos.
